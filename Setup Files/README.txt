This is fully functional Demo App for renting car


Instruction for Installation:

1.Turn off Anti-Virus or windows defender.
2.download the Setup Files.rar file.
3.download and install xampp and move the flutter file to 'xampp/htdocs' for API connection.
4.go to http://localhost/phpmyadmin/ and create a Database and import the rentalcar sql file.
5.run the rentalcar_setup.exe file.
6.enjoy!
